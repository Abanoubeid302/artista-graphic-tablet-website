# 🎨 Artista – Graphic Tablets Store
A modern responsive landing page for a graphic tablets store built using HTML and CSS.

🔥 Project Overview

Artista is a simple front-end project designed to practice layout design and responsive UI using Flexbox and CSS Grid. The focus was on building a clean structure and modern visual style.

🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)

✨ Features

- Responsive layout
- Modern design using gradients
- Flexbox-based structure
- CSS Grid for page layout
- Clean and simple UI
